PREREQUISITES
League of Legends Client opened
Client Resolution 1600x900
Client navigated to the 'Loot / Champions' section
Java installed

Using SikuliX IDE
* download and 'install' sikulix from https://raiman.github.io/SikuliX1/downloads.html
* load the league_essence.sikuli script
* Run the script (play button)
* To stop the script, use the default shortcut ctrl + shift + c

Using the .jar file league_essence_sikuli.jar directly
* run the jar file (doubleclick or from command line)
* to end, press ctrl + shift + c, if it does not end, repeat OR launch task manager and kill java instances OR put any other window in front of the client so the script is unable to find the images and crashes / times out OR restart PC
